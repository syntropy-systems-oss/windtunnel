# Support

Wind Tunnel is community-supported open-source software. Maintenance and issue
responses are provided on a best-effort basis; using the project does not create
a support contract or service-level agreement with Syntropy Systems, Inc.

Before opening an issue:

1. Check the [documentation](https://syntropy-systems-oss.github.io/windtunnel/)
   and existing issues.
2. Confirm the problem still occurs on the latest release.
3. Reduce it to a minimal reproduction that does not contain credentials,
   customer information, or private trace data.

Use a GitHub issue for reproducible bugs, focused feature proposals, and unclear
documentation. General integration work is welcome when it can improve the
public documentation for other users too.

Security vulnerabilities must be reported privately according to
[`SECURITY.md`](SECURITY.md), not through a public issue.
